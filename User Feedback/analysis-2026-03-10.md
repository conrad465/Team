## Findings

### Bucket 1 — Rendering / Engine Bugs
Observed across tc_001 and tc_002. These are engine-level issues unrelated to the system prompt.

- **Label positioning**: Labels render too far from their reference elements (tc_001: price label; tc_002: condensation label)
- **Element collision**: Elements overlap incorrectly (tc_001: equilibrium point collides with supply arrow)
- **Bounds overshoot**: Arrows extend past the canvas boundary (tc_001: supply/demand arrows past bottom)
- **Axis intersection bug**: Graph axes intersect each other rather than meeting at a point (tc_001)
- **Arrow animation jitter**: Arrows animate partway, partially revert, then complete — visually jarring (tc_002)

### Bucket 2 — Content Quality
Observed across tc_003, tc_004, tc_005. Likely an artifact of an outdated system prompt.

- Visualizations restate the voiceover in boxes rather than adding to understanding
- Not additive: a viewer learns nothing from the visual that they wouldn't get from audio alone
- **Important caveat**: tester noted on tc_005 that these scenes were generated before the system prompt was updated. This bucket may already be resolved.

## Recommended Next Step

Regenerate tc_001–tc_005 with the current system prompt and re-test before fixing anything. This will:
1. Confirm whether Bucket 2 (content quality) is already resolved
2. Confirm whether Bucket 1 (rendering bugs) reproduces reliably on fresh scenes

If Bucket 2 persists after regeneration → system prompt work needed.
If Bucket 1 persists after regeneration → dev work needed on positioning/animation engine.
